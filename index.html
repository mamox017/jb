About The Job Board
The Job Board is a website written in the HTML Bootstrap framework along with JavaScript. It utilizes the Firebase Firestore database technology in order to keep track of data about job listings and other relevant information. Firebase authentication is used in order to store user login credentials and manage users on the website.
